Sobre a Linguagem

HTML (HyperText Markup Language) é a linguagem fundamental para criar e estruturar páginas webs. Desde sua criação o HTML evoluiu consideravelmente, passando por diversas versões, sendo o HTML5 que é a mais recente delas.

O que são as Tags

As tags são usadas para que o site fique estruturado, existem vários tipos de tags:


Marked text

As tags podem ser usadas para destacar partes importantes do texto, como este exemplo usando a tag .

Small text

A tag é usada para texto pequeno, como notas ou legendas.

Italicized text

A tag é usada para texto em itálico.

Underlined text

A tag é usada para texto sublinhado.

Strong text

A tag é usada para texto forte ou enfatizado.

Abbr

A tag é usada para abreviações, e você pode ver a expansão quando passar o mouse sobre ela.

Font text

A tag é usada para especificar o tamanho e a cor da fonte.
